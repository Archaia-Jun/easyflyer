# Stack :

- firestore, Firebase CLI

- node.js, npm, express, javascript, typescript

- Angular

- NVM,

- Postman , curl

- Git

- Visual Paradigm



IDE : Visual Studio Code

- Storing Datas in database,

- Carts populated with items , prices and quantities,

- Cloud functions (Google Cloud Platform)
